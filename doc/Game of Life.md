# 生命游戏

“生命游戏”（**Conway’s Game of Life**）是数学家约翰·康威（John Conway）在1970年提出的一个**元胞自动机（Cellular Automaton）**模型，用于模拟“生命”的演化过程。
它虽然规则简单，但能产生非常复杂的动态行为。

---

## 🎮 一、基本设定

* 世界是一个 **二维网格**（通常为无限大，但计算机实现时是有限的）。
* 每个格子（称为“细胞”cell）只有两种状态：

  * **存活（Alive）**
  * **死亡（Dead）**
* 每个细胞的状态在下一回合由**周围八个邻居**（上下左右和四个对角线）共同决定。

---

## 🧬 二、规则（简称 “B3/S23”）

规则可以总结为：

| 当前状态 | 邻居数量      | 下一状态 | 解释         |
| ---- | --------- | ---- | ---------- |
| 存活   | 2 或 3     | 继续存活 | 有足够的“生命支持” |
| 存活   | < 2 或 > 3 | 死亡   | 太孤单或太拥挤    |
| 死亡   | 恰好 3      | 复活   | “繁殖”产生新生命  |
| 死亡   | 其他数量      | 继续死亡 | 没有足够条件复活   |

简写成：

> **Birth 3 / Survival 23** → **B3/S23**

---

## 🌱 三、演化过程

1. 设定初始状态（任意组合的活细胞）。
2. 同时更新所有细胞到下一代（基于当前状态和规则）。
3. 重复迭代，观察形态的变化。
